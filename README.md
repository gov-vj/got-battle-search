# got-battle-search

Application url: [https://warm-coast-80493.herokuapp.com](https://warm-coast-80493.herokuapp.com)

### Installation
```
git clone git@github.com:gov-vj/got-battle-search.git
cd got-battle-search
npm install
npm run build
npm run serverstart
```
